# OneShotpad
## Description

A 6-key productivity macropad inspired by the atmospheric aesthetic of OneShot.
Designed as a compact command-line companion and workflow accelerator.

## Features
Project Type: Custom Hackpad
Firmware: KMK (CircuitPython)
MCU: Seeed Studio XIAO RP2040
Case: Custom OneShot-inspired enclosure
Display: 0.96" I2C OLED

## BOM (Bill of Materials)

| Qty | Part | Description |
|-----|------|-------------|
| 1   | Seeed XIAO RP2040 | MCU with USB-C and GPIO pins |
| 6   | MX Mechanical Switches | 4x4 grid layout |
| 6   | Keycaps | Compatible with MX switches |
| 1   | 0.96" OLED Display | I2C 128x68 display for status/animations |
| 2   | 3D Printed Parts | Top and bottom case halves |
| 4   | M3x16mm Screws | Case assembly |
| 4   | Heatset Inserts | For threaded screw holes in the case |
| 1   | PCB | 3x2 keypad matrix with diodes and LED traces |
